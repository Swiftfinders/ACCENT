# ACCENT Website - Image Sources Guide

This document provides detailed guidance on sourcing professional stock images for the ACCENT premium website.

## Image Requirements

All images should be:
- **High resolution**: Minimum 1920px wide for hero/backgrounds, 800px for portfolio items
- **Professional quality**: Sharp, well-lit, luxury aesthetic
- **Optimized for web**: Use tools like TinyPNG or ImageOptim to compress before deployment
- **Properly licensed**: Use free stock sites or properly licensed images

---

## Recommended Stock Photo Sources

### Free High-Quality Sources:
1. **Unsplash** (https://unsplash.com) - Premium quality, free for commercial use
2. **Pexels** (https://pexels.com) - High-quality stock photos and videos
3. **Pixabay** (https://pixabay.com) - Large collection of free images

### Premium Sources (if budget allows):
1. **Shutterstock** (https://shutterstock.com)
2. **Adobe Stock** (https://stock.adobe.com)
3. **iStock** (https://istockphoto.com)

## Image Optimization Guide

### Before Upload:
1. **Resize**: Use appropriate dimensions
   - Hero: 1920x1080px or larger
   - Portfolio: 1200x900px
   - About: 1000x1200px

2. **Compress**: Use online tools
   - TinyPNG (https://tinypng.com)
   - ImageOptim (Mac app)
   - Squoosh (https://squoosh.app)

3. **Format**:
   - Use JPEG for photographs
   - Aim for 80-85% quality
   - File size target: <300KB per image

### File Naming:
- Use lowercase
- Use hyphens, not spaces
- Be descriptive
- Match the references in HTML

## Quick Download Instructions

### Using Unsplash:

1. Go to https://unsplash.com
2. Search using terms above
3. Find high-quality image matching description
4. Click image → Click "Download free" button
5. Rename file to match required filename
6. Optimize image (compress)
7. Place in appropriate folder

### Using Pexels:

1. Go to https://pexels.com
2. Search using terms above
3. Select image → Click "Download"
4. Choose "Original" or "Large" size
5. Rename and optimize
6. Place in appropriate folder
